# Editkaro Portfolio
A modern and responsive portfolio website built using **React + TypeScript**, powered by **Vite** and styled with **Tailwind CSS**. The project is optimized for performance and supports seamless deployment on **GitHub Pages**.
---
## Features
* Responsive and mobile-friendly design
* Modern hero section with branding
* Featured projects and services showcase
* Interactive video gallery section
* Client testimonials section
* Contact form interface
* Smooth navigation and clean footer
* Fast build and optimized performance
---
## Tech Stack
* **React JS**
* **TypeScript**
* **Vite**
* **Tailwind CSS**
* **GitHub Pages** (via `gh-pages`)
---
## Project Scripts
| Command           | Description                      |
| ----------------- | -------------------------------- |
| `npm run dev`     | Start local development server   |
| `npm run build`   | Create production build          |
| `npm run preview` | Preview production build locally |

## Installation & Setup
### 1.Clone the Repository
```bash
git clone <repository-url>
```
### 2.Install Dependencies
```bash
npm install
```
### 3.Start Development Server
```bash
npm run dev
```
### 4.Build for Production
```bash
npm run build
```
### 5.Deploy to GitHub Pages
```bash
npm run deploy
```
---
## Deployment
This project is configured for deployment using **GitHub Pages**.
Before deploying:
* Ensure the `homepage` field in `package.json` is set correctly.
* Verify that the GitHub repository remote URL is properly configured.
* The production build is generated inside the `dist` folder and published using `gh-pages`.
Example:
```json
"homepage": "https://yourusername.github.io/repository-name/"
```
